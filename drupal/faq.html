FAQ

  1. Button installs which version of Drupal?

    Button will install the latest version of Drupal 8 with MySQL & PostgreSQL.

  2. How do I upgrade Drupal with the latest version from admin section?

    There is not going to be any way to upgrade Drupal from admin section or automatically by pushing a button because of the Heroku file system limitations. You need to first clone your repository then download the latest version of Drupal from Download Drupal then push all the changes on Heroku App using git.

  3. How do I upgrade Drupal Plugins with the latest version?

    There is not going to be any way to upgrade Drupal Plugins from admin section because of the Heroku file system limitations. You need to first clone your repository then download the latest version of your plugins then merge the plugin folders with existing plugins then next push the plugin folders using git.
